Get Started

This guide explains how to set up your Angular project to begin using Mozek, a high-performance, signal-based component library.

This guide assumes that the Angular CLI has already been installed.



Install Mozek

Add Angular Material to your application by running the following command:

npm install @mozek-angular

@if (textCopied('copyNpm')) { check_circle} @else { copy}

You're done! Mozek is now configured to be used in your application.



Import a Component

To use a Mozek component, simply import it into your Angular component:

import { MozButton } from 'mozek-angular';

@if (textCopied('copyImport')) { check_circle} @else { copy}

Then, add the component to your template:

<moz-button>Click Me</moz-button>

@if (textCopied('copyTag')) { check_circle} @else { copy}


Run Your Application

To run your application, use the following command:

ng serve

@if (textCopied('copyServe')) { check_circle} @else { copy}

Open your browser and navigate to http://localhost:4200 to see Mozek in action!
