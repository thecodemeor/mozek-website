**Mozek Website** is the official documentation and showcase platform for the Mozek UI Library.
It serves as a comprehensive reference for developers to explore, understand, and implement Mozek UI components in real projects.

The website demonstrates component usage, design tokens, theming options, and best practices through interactive examples and clear documentation. It is designed to be the single source of truth for everything related to Mozek UI, from getting started to advanced customization.

**Mozek Website** aims to ensure consistency, usability, and clarity by presenting real world implementations alongside guidelines and code examples, enabling developers to build modern and maintainable user interfaces with confidence.
